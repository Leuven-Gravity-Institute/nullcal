# Welcome to nullcal's documentation

`nullcal` is a Python package for constraining calibration errors of a
closed-geometry network of gravitational-wave detectors.

## Modules

- `clustering` — Clustering algorithms for time-frequency maps
- `likelihood` — Likelihood computations for recalibration
- `metadata` — YAML metadata handling
- `null_stream` — Core null stream logic (projection, whitening, calibration)
- `prior` — Prior definitions
- `result` — Result analysis and plotting
- `time_frequency_transform` — Wavelet and STFT transforms
- `utils` — SNR, logging, geometry utilities
